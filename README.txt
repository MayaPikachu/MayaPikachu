# Compilando

use o comando
```
g++ trab_4.cpp -o trab_4.exe
```
para compilar o programa.


# Executando

Para executar o programa, use `./trab_4.exe caminho/pro/arquivo`.


# Resultados 

Os resultados são escritos nos arquivos "Info_ABB.txt" e "Info_AVL.txt".
As 10 palavras mais frequentes são escritas para a saída padrão.
